# Reader Tests

This directory contains unit tests for the reader components.

## Files

*   **`ReaderView.test.tsx`**: Tests for the main `ReaderView` component, often involving mocking `epub.js`.
*   **`AnnotationList.test.tsx`**: Tests for the annotation sidebar list.
*   **`AnnotationPopover.test.tsx`**: Tests for the annotation creation popup.
*   **`GestureOverlay.test.tsx`**: Tests for gesture recognition logic.
*   **`TTSQueue.test.tsx`**: Tests for the TTS queue rendering and interactions.
*   **`TTSQueue_AutoScroll.test.tsx`**: Specific tests verifying the auto-scroll behavior of the TTS queue to keep the active sentence in view.
