# PRD: Device-Optimized Experiences

**Status:** Draft | **Priority:** Major | **Category:** UX

## 1. Problem Statement

One UI doesn't fit all devices:
- Phone: One-handed, small targets
- Tablet: Two-handed, reading focus
- Desktop: Mouse, multi-window
- Car: Large buttons, voice
- Watch: Minimal, controls only

Currently, same UI everywhere = suboptimal everywhere.

## 2. Vision

**Right experience for right device**:
- Phone: Immersive reading, easy TTS access
- Tablet: Split-view, annotation sidebar
- Desktop: Library management, bulk operations
- Car Mode: Giant buttons, voice commands
- Watch: Play/pause, skip, basic controls
