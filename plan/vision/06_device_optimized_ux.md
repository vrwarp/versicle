# UX: Device-Optimized Experiences

**PRD:** [06_device_optimized_prd.md](06_device_optimized_prd.md)

## Phone Layout

- Single column, full immersion
- Bottom bar: Play, progress, quick settings
- One-handed reachability focus

## Tablet Layout

- Reading pane (60%) + sidebar (40%)
- Sidebar: TOC, annotations, search
- Apple Pencil for highlighting

## Desktop Layout

- Library: Multi-column grid
- Keyboard shortcuts (j/k for nav, space for play)
- Multi-window support
