# PRD: TTS v2 Completion

**Status:** Draft | **Priority:** Incremental | **Category:** Audio

## 1. Overview

Complete the remaining TTS v2 features for a premium listening experience across all devices.

## 2. Feature Summary

| Feature | Effort | Device Benefit |
|---------|--------|----------------|
| **Sleep Timer** | Low | All: Evening use |
| **Smart Speed** | Medium | All: More listening in less time |
| **Car Mode** | Medium | Phone: Safer controls |
| **Chapter Pre-roll** | Low | All: Context on transitions |
| **Earcon Feedback** | Low | All: Audio cues for actions |
| **Voice Switching** | High | All: Immersion |
| **Ambient Sounds** | Medium | All: Focus |
| **MP3 Export** | Medium | All: Offline portability |

## 3. Recommended Order

**Wave 1 (Quick Wins):** Sleep Timer, Earcons, Chapter Pre-roll
**Wave 2 (Core):** Audio Graph, Smart Speed, Car Mode  
**Wave 3 (Delight):** Voice Switching, Ambient Sounds, MP3 Export

## 4. Multi-Device Considerations

| Feature | Cross-Device Sync |
|---------|-------------------|
| Sleep Timer | Local only (per-session) |
| Smart Speed | Setting syncs, state local |
| Car Mode | Auto-detect, no sync |
| Earcons | Volume syncs |
| Voice prefs | Full sync |
